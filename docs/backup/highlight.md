# 提示与警告

系统中提供了一些基本的提示与警告以来丰富文档的可读性。

## 提示与警告标签

``` html
<div class="book-tip">
This is a tip
</div>

<div class="book-warning">
This is a warning
</div>

<div class="book-danger">
This is a dangerous thing
</div>
```
